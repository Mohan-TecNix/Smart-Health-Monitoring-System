# Smart Health Monitoring System using ESP32

## Overview
A simple health monitoring system built using ESP32 that monitors:

- Heart Rate (Simulated using Potentiometer)
- Body Temperature (DS18B20)
- OLED Display Output
- Buzzer Alert System

## Components Used
- ESP32 DevKit V1
- OLED SSD1306 Display
- DS18B20 Temperature Sensor
- Potentiometer
- Active Buzzer

## Features
- Real-time monitoring
- OLED display visualization
- Alert indication using buzzer
- Wokwi simulation support
